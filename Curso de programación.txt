🚀 DÍA 1 — Setup del Proyecto
----------------------------
Objetivo del día
Tener el proyecto estructurado, el entorno virtual funcionando y el primer commit en Git.

Paso 1 — Crea la estructura de carpetas
Abre tu terminal y ejecuta:

# 1. Crear carpeta raíz
mkdir tienda
cd tienda

# 2. Crear subcarpetas
mkdir data
mkdir src

# 3. Crear archivos vacíos iniciales
touch main.py
touch requirements.txt
touch README.md
touch src/__init__.py
touch src/productos.py
touch src/persistencia.py

Resultado: Carpeta del proyecto con la estructura base creada ✅



Paso 2 — Entorno Virtual
------------------------

# Crear entorno virtual
python -m venv .venv
# En Mac OS debes usar python3 -m venv .venv

# Activarlo (Linux/Mac)
source .venv/bin/activate

# Activarlo (Windows)
.venv\Scripts\activate

# Verificar que está activo — debe mostrar la ruta del .venv
which python

💡 ¿Por qué un entorno virtual? Aísla las dependencias de tu proyecto.
Es una práctica profesional desde el día 1. Cada proyecto tiene sus propias
librerías sin contaminar otros.

Resultado: Entorno virtual activo con Python 3.13.13 ✅



Paso 3 — .gitignore
-------------------
Crea el archivo .gitignore con este contenido:

# Crear el archivo .gitignore
touch .gitignore

## Copiar el siguiente contenido en el archivo .gitignore
# Entorno virtual
.venv/

# Python
__pycache__/
*.pyc
*.pyo

# Datos sensibles o generados
data/*.json

# Editor
.vscode/
.idea/

Resultado: Archivo .gitignore creado con el contenido ✅

Paso 5 — Crear el README.md
--------------------------
¿Qué es un README.md?
Es la primera página de tu proyecto. Cualquier persona que vea tu código entiende de qué trata,
cómo instalarlo y cómo usarlo.

💡 Regla profesional — Un proyecto sin README es un proyecto incompleto.
GitHub lo muestra automáticamente como portada del repositorio.

¿Qué es Markdown?
El .md significa Markdown — es un lenguaje simple para dar formato a texto plano:

# Título grande
## Título mediano
### Título pequeño

**negrita**
*cursiva*

- elemento de lista
- otro elemento

Nota. No necesitas memorizarlo ahora, lo irás aprendiendo naturalmente.
